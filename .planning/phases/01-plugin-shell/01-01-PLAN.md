Create three files that constitute the entire plugin shell. All code patterns are specified below -- follow them exactly.

**CMakeLists.txt** -- Root CMake file:
- cmake_minimum_required(VERSION 3.22), project(HarmonicOscillator VERSION 0.1.0)
- C++17 standard required
- FetchContent_Declare JUCE from https://github.com/juce-framework/JUCE.git, GIT_TAG 8.0.12, GIT_SHALLOW TRUE
- juce_add_plugin with:
  - COMPANY_NAME "Die stille Erde"
  - IS_SYNTH TRUE, NEEDS_MIDI_INPUT TRUE, NEEDS_MIDI_OUTPUT FALSE, IS_MIDI_EFFECT FALSE
  - EDITOR_WANTS_KEYBOARD_FOCUS FALSE
  - COPY_PLUGIN_AFTER_BUILD TRUE
  - PLUGIN_MANUFACTURER_CODE DsEr
  - PLUGIN_CODE Hrmo
  - FORMATS AU VST3 Standalone
  - PRODUCT_NAME "Harmonic Oscillator"
- target_sources: src/PluginProcessor.cpp only (no separate editor files)
- target_compile_definitions: JUCE_WEB_BROWSER=0, JUCE_USE_CURL=0, JUCE_VST3_CAN_REPLACE_VST2=0, JUCE_DISPLAY_SPLASH_SCREEN=0
- target_link_libraries: PRIVATE juce::juce_audio_utils, PUBLIC juce::juce_recommended_config_flags, juce::juce_recommended_lto_flags, juce::juce_recommended_warning_flags

**src/PluginProcessor.h** -- AudioProcessor subclass:
- Include JuceHeader.h
- Class PluginProcessor : public juce::AudioProcessor
- Public members: constructor, destructor, prepareToPlay, releaseResources, processBlock, createEditor, hasEditor (returns true), getName (returns JucePlugin_Name), acceptsMidi (true), producesMidi (false), isMidiEffect (false), getTailLengthSeconds (0.0), getNumPrograms (1), getCurrentProgram (0), setCurrentProgram (empty), getProgramName (empty string), changeProgramName (empty), getStateInformation, setStateInformation
- Public member: juce::AudioProcessorValueTreeState apvts
- Private: static createParameterLayout(), JUCE_DECLARE_NON_COPYABLE_WITH_LEAK_DETECTOR

**src/PluginProcessor.cpp** -- Full implementation:

Constructor uses member initializer list:
- Base: AudioProcessor(BusesProperties().withOutput("Output", juce::AudioChannelSet::stereo(), true))
- apvts: (*this, nullptr, juce::Identifier("HarmonicOscillatorParams"), createParameterLayout())

createParameterLayout() -- static function returning ParameterLayout:
- Helper lambda `skewedRange` for NormalisableRange with skew = 1.0f / 3.0f (NOT 3.0f -- this is the inverse; gives more resolution at low values)
- 6 AudioProcessorParameterGroup sections:

  1. **Pitch group** ("pitch", "Pitch"):
     - AudioParameterInt: "coarse_tune" / "Coarse Tune", -24 to 24, default 0
     - AudioParameterFloat: "fine_tune" / "Fine Tune", range -100 to 100 (step 0.01), default 0
     - AudioParameterFloat: "glide" / "Glide", skewedRange(0, 2000), default 0, label "ms"

  2. **Harmonics group** ("harmonics", "Harmonics"):
     - Loop i = 1..8: AudioParameterFloat "harm_{i}" / "Harmonic {i}", range 0-1, default 1.0 for i==1, 0.0 otherwise

  3. **Scan/Tilt group** ("scan", "Scan/Tilt"):
     - AudioParameterFloat: "scan_center" / "Scan Center", range 0-1, default 0.5
     - AudioParameterFloat: "scan_width" / "Scan Width", range 0-1, default 1.0
     - AudioParameterFloat: "spectral_tilt" / "Spectral Tilt", range -1 to 1, default 0

  4. **FM group** ("fm", "FM"):
     - AudioParameterFloat: "lin_fm_depth" / "Lin FM Depth", range 0-1, default 0
     - AudioParameterFloat: "exp_fm_depth" / "Exp FM Depth", range -1 to 1, default 0
     - AudioParameterFloat: "fm_rate" / "FM Rate", skewedRange(0.1, 20000), default 1.0, label "Hz"

  5. **Envelope group** ("envelope", "Envelope"):
     - AudioParameterFloat: "attack" / "Attack", skewedRange(1, 2000), default 10, label "ms"
     - AudioParameterFloat: "release" / "Release", skewedRange(1, 5000), default 100, label "ms"

  6. **Output group** ("output", "Output"):
     - AudioParameterFloat: "master_level" / "Master Level", range 0-1, default 0.8
     - AudioParameterChoice: "output_select" / "Output Select", choices ["Harmonic Mix", "Triangle", "Sawtooth", "Square"], default index 0

  All ParameterIDs use version 1: juce::ParameterID { "param_id", 1 }

processBlock: just call buffer.clear() (no DSP in Phase 1). Comment the unused midiMessages parameter.

prepareToPlay / releaseResources: empty stubs.

createEditor: return new juce::GenericAudioProcessorEditor(*this)

getStateInformation:
- auto state = apvts.copyState()
- create XML from state
- copyXmlToBinary(*xml, destData)

setStateInformation:
- getXmlFromBinary to get XML
- null-check XML and verify tag name matches apvts.state.getType()
- apvts.replaceState(ValueTree::fromXml(*xmlState))

createPluginFilter: return new PluginProcessor() (free function at bottom of .cpp)

IMPORTANT: Do NOT create separate PluginEditor.h/PluginEditor.cpp files. The GenericAudioProcessorEditor is returned directly from createEditor() in the processor -- no wrapper needed.
