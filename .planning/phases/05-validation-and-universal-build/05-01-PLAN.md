<task type="auto">
  <name>Task 1: Universal binary build, pluginval, and auval validation</name>
  <files>CMakeLists.txt</files>
  <action>
1. Edit CMakeLists.txt: Add two lines BEFORE the `project(HarmonicOscillator ...)` line (line 2):
   ```cmake
   set(CMAKE_OSX_ARCHITECTURES "arm64;x86_64")
   set(CMAKE_OSX_DEPLOYMENT_TARGET "11.0")
   ```
   These must appear before `project()` so CMake picks them up for all targets.

2. Clean rebuild:
   ```bash
   rm -rf build
   cmake -B build -DCMAKE_BUILD_TYPE=Release
   cmake --build build --config Release
   ```

3. Verify universal binary with lipo on all three artefacts:
   ```bash
   lipo -archs "build/HarmonicOscillator_artefacts/Release/VST3/Harmonic Oscillator.vst3/Contents/MacOS/Harmonic Oscillator"
   lipo -archs "build/HarmonicOscillator_artefacts/Release/AU/Harmonic Oscillator.component/Contents/MacOS/Harmonic Oscillator"
   lipo -archs "build/HarmonicOscillator_artefacts/Release/Standalone/Harmonic Oscillator.app/Contents/MacOS/Harmonic Oscillator"
   ```
   Each must output: `x86_64 arm64`

4. Download pluginval:
   ```bash
   curl -L "https://github.com/Tracktion/pluginval/releases/download/latest_release/pluginval_macOS.zip" -o /tmp/pluginval.zip
   unzip -o /tmp/pluginval.zip -d /tmp/pluginval
   ```

5. Run pluginval at strictness level 10 for VST3 (no trailing slash on .vst3 path):
   ```bash
   /tmp/pluginval/pluginval.app/Contents/MacOS/pluginval --strictness-level 10 --validate "build/HarmonicOscillator_artefacts/Release/VST3/Harmonic Oscillator.vst3"
   ```

6. Run pluginval at strictness level 10 for AU:
   ```bash
   /tmp/pluginval/pluginval.app/Contents/MacOS/pluginval --strictness-level 10 --validate "build/HarmonicOscillator_artefacts/Release/AU/Harmonic Oscillator.component"
   ```

7. Run auval for the AU component (type=aumu, subtype=Hrmo, manufacturer=DsEr):
   ```bash
   auval -v aumu Hrmo DsEr
   ```
   Must report "validation successful" or equivalent pass status.

Note: pluginval at strictness > 5 runs auval internally too, but the explicit auval run confirms registration and provides Apple's official validation output.

  <how-to-verify>
Load the Harmonic Oscillator plugin and the Max/MSP gen~ patch side by side. Compare output at 44.1kHz (repeat at 48kHz and 96kHz if convenient):

1. **Harmonic mix**: Set several harmonic levels (e.g., H1=1.0, H3=0.7, H5=0.5, rest=0), play a sustained note. Compare timbre.
2. **Scan sweep**: Set scan width to ~0.3, slowly sweep scan center 0 to 1. Compare how harmonics fade in/out.
3. **Tilt extremes**: Set tilt to +24dB (bright), then -24dB (dark), then 0dB (flat). Compare spectral balance.
4. **PolyBLEP waveforms**: Switch output to Saw, Square, Triangle in turn. Play across octaves (C2, C4, C6). Compare waveform character.
5. **FM**: Enable linear FM, sweep LFO rate. Compare modulation character.

Expected: Output should be perceptually identical or very close. Minor floating-point differences are acceptable if not audible.
  </how-to-verify>
